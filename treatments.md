## Treatment Philosophy

### Why Parallel > Sequential
Standard oncology: Try A → fails → try B → fails → try C
Sid's approach: Try A + B + C + D simultaneously → measure what works → adjust

### The Layered Attack Model
Each modality targets a different vulnerability:

```
Checkpoint inhibitors ──→ Remove immune brakes
Neoantigen vaccines   ──→ Train immune recognition
Oncolytic virus       ──→ Kill directly + release antigens
NK cells              ──→ Innate immune killing
Radioligand therapy   ──→ Targeted radiation to FAP+ cells
XGeva                 ──→ Protect bone architecture
GM-CSF / Anktiva      ──→ Amplify immune response
```

The combination created a synergistic effect: the oncolytic virus and radioligand killed tumor cells, releasing antigens. The vaccines had already primed T cells against those antigens. Checkpoint inhibitors ensured T cells could act. The result was a transformed immune microenvironment.
